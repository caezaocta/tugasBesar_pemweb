Catatan modifikasi
==================


[ ./auth.php ]

Menambahkan value 'admins' yang merupakan daftar *USERNAME* dari user
yang bertindak sebagai admin di aplikasi.

Untuk mendefinisikan nama-nama admin, gunakan format sebagai berikut
di dalam file .env:

    APP_ADMINS='nama_admin_1,nama_admin_2,...'

    (dipisahkan oleh tanda koma, dan tidak boleh ada spasi)
